# Contributors to Demiarch

Demiarch is a community project. Thank you to all contributors who have helped shape this project.

## Core Maintainers

- [Demiarch Contributors](https://github.com/demiarch/demiarch) - Initial development and ongoing maintenance

## Contributors

We use GitHub's [Contributors](https://github.com/demiarch/demiarch/graphs/contributors) list to track contributions automatically.

### Recognition

Contributions are recognized in multiple ways:
- Listed in this file
- Cited in release notes
- Mentioned in CHANGELOG.md for significant contributions
- Listed on GitHub's contributors page

## Contribution Types

We recognize various types of contributions:

### Code Contributions
- New features
- Bug fixes
- Performance improvements
- Refactoring
- Testing

### Documentation Contributions
- README improvements
- Documentation pages
- Examples and tutorials
- Code comments
- Translation

### Community Contributions
- Bug reports
- Feature requests
- Code reviews
- Issue triage
- Answering questions
- Community moderation

## How to Get Listed

Any contribution to the project qualifies you for recognition in this file:

1. **Code or Documentation**: Submit a pull request that gets merged
2. **Issues and Discussions**: Provide valuable feedback that improves the project
3. **Community Support**: Help other users with their questions

**Note**: This file is periodically updated to include new contributors. If you've contributed but don't see your name, please open an issue or PR to add it.
